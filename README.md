# SimpleAnki API

## Description
SimpleAnki API is a backend service built with ASP.NET Core. It provides RESTful endpoints for managing flashcards and decks for a spaced-repetition learning application.

---

## Technologies
- ASP.NET Core 9
- Entity Framework Core  
- SQL Server / SQLite  
- RESTful API design  

---
